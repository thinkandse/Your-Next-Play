# Your-Next-Play
Your Next Play - Refer to Read Me, for Year 10 Digital Technologies.

Steps to run:
 - Run the app.py file - this should automatically run every html file as well.

 - Database steps:
  - SQLiteStudio is recommended, under Your-Next-Play -> Tables -> users, there should be a list of users under data section with password, username, and email.

You may run on various different programs, but Pycharm or VS Code are recommended.

The files in the folders (static & template), or the ones that have 'FINAL' are the most updated ones, refer to those for the best website.

Uses HTML, Jinja, Python, Javascript, CSS, and different fonts & images.
